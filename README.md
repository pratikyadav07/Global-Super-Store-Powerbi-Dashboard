# Global Superstore Power BI Dashboard

This project presents an interactive sales analysis dashboard built using Power BI based on the Global Superstore dataset.

The dashboard provides insights into sales performance, profit trends, customer segments, and regional performance across different product categories.

## Tools & Technologies

* Microsoft Power BI
* Data Cleaning
* Data Visualization
* CSV Dataset

## Key Insights

* Total sales and profit analysis
* Region-wise sales performance
* Category and sub-category performance
* Customer segment analysis
* Monthly sales trends

## Dataset

The dataset used in this project is the Global Superstore dataset which contains sales transactions, product categories, customer segments, and regional sales data.

## Project Objective

The objective of this project is to analyze business sales data and transform it into meaningful insights using interactive dashboards.
